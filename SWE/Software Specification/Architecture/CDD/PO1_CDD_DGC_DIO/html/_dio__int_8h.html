Description

This function is used to change the Software pin value
Parameters
InputCopy_u8SwPinNo , Copy_u8PinVal
OutputN/A
Input/OutputN/A
Returns

- E_OK
- E_NOK
Main Steps

1- Determine the pin ID and the port ID from the the SW pin number passed to the function as a parameter.
2- Using the PORT ID a switch case is made to assign the desired mode to the desired pin in the right port
3- There are 2 values in which a pin can be changed to (HIGH and LOW).
4- For HIGH Value the PORTx register is set to 1 at the bit corresponding to the pin ID.
5- For LOW Value the PORTx register is set to 0 at the bit corresponding to the pin ID.
