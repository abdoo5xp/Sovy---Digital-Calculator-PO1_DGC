Req_PO1_DGC_GDD_PORT_001-V1.1
Description

This API function that is responsible for initializing all pins and ports to their operation mode and initial values.
Parameters
Inputvoid
OutputN/A
Input/OutputN/A
Returns
: void
Main Steps

1- Check for the initial mode (ex:PORT_u8PIN_0_INIT_MODE) assigned to each pin in the configuration file
2- un-define the default value and direction for each pin (ex: PORT_u8PIN_0_DEFAULT_VALUE & PORT_u8PIN_0_DEFAULT_DIR)
3- re-define the default value and direction for each pin as stated in the configuration file
4- If one of the pins is not configured correctly a warning message shall appear and then the default value of this pin shall be set to low and the direction to output.
5- finally, using the CONC function each port shall be set with the corresponding values and directions previously set in the configuration file. (CONC function shall place the defined value in both DDRx and PORTx registers)

Description

This function is used to change the Software pin Mode
Parameters
InputCopy_u8SwPinNo , Copy_u8PinDir
OutputN/A
Input/OutputN/A
Returns

- E_OK
- E_NOK
Main Steps

1- Determine the pin ID and the port ID from the the SW pin number passed to the function as a parameter.
2- Using the PORT ID a switch case is made to assign the desired mode to the desired pin in the right port
3- There are 2 directions in which a pin can be changed to (output and input).
4- For output direction of the desired pin the DDR register is set to high at the bit corresponding to the pin ID.
5- For input direction of the desired pin the DDR register is set to 0 at the bit corresponding to the pin ID.

Description

This function is used to change the Software pin Mode
Parameters
InputCopy_u8SwPinNo , Copy_u8PinMode
OutputN/A
Input/OutputN/A
Returns

- E_OK
- E_NOK
Main Steps

1- Determine the pin ID and the port ID from the the SW pin number passed to the function as a parameter.
2- Using the PORT ID a switch case is made to assign the desired mode to the desired pin in the right port
3- There are 4 modes in which a pin can be set to (output low, output high, input high impedance, input pull up).
4- For the 2 output modes the DDR register is set to high at the bit corresponding to the pin ID, and the PORT register is set to 1 for HIGH and 0 for LOW.
5- For the 2 input modes the DDR register is set to 0 at the bit corresponding to the pin ID, and the PORT register is set to 1 in case of PULL UP option and 0 for the HIGH IMPEDANCE option.
